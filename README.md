# Mini Calculator

A simple calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Decimal input
- Clear button
- Sign toggle (`±`)
- Percentage (`%`)

## Run

Open `index.html` in a browser to use the calculator.

## Files

- `index.html` — calculator structure
- `style.css` — visual layout and styling
- `script.js` — button creation and calculator logic
